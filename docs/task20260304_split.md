  - **Amazon**: Amazon主图风格，纯白色背景RGB 255 255 255，[商品名称]单体产品展示，无任何文字LOGO水印，产品居中填充画面85%以上，光影自然柔和，高端质感摄影，极简主义，专业商业产品照，干净留白，写实风格，超高细节，2000px正方形，8k清晰
  - **Etsy**: Etsy风格生活场景图，[商品名称]自然融入日常，文艺氛围，温暖自然光，欧美模特/手作感，故事性构图，柔和色调，ins风摄影，背景虚化生活环境如咖啡馆/海边/家居，高端质感，无文字干扰，电影感光影，8k高清

- **涉及页面**: StudioGenesisForm
- **备注**: 无

### 6. [feat] AI 优化提示词按钮
- **优先级**: P1
- **描述**:
  - 在「组图要求」文本框右下角放置 AI 图标按钮（sparkle icon）
  - 点击后根据当前平台做 SEO 关键词优化：

- **涉及页面**: StudioGenesisForm
- **备注**: 需要后端 Chat API 支持

### 7. [feat] 生成结果持久保留
- **优先级**: P1
- **描述**:
  - 每次生成后，新结果追加到右侧「生成结果」区域，不覆盖之前图片
  - 浏览器标签页关闭前，所有已生成图片持续保留
  - 以缩略图网格展示，标注批次序号或生成时间
- **涉及页面**: StudioGenesisForm
- **备注**: 使用前端状态管理（Zustand store）维护会话级历史

### 8. [feat] 生成结果原位编辑模式
- **优先级**: P1
- **描述**:
  取消「批量编辑」跳转，改为原位直接编辑：
  1. 点击图片 → 放大展示 + 背景虚化遮罩
  2. 放大图下方弹出提示词输入框，预填当前图的生成提示词
  3. 修改提示词后点击「重新生成」→ 基于当前图重新生成
  4. 新图同样支持继续点击编辑、再次重新生成（可反复迭代）
  5. 满意后点击「保存」保存编辑状态
  6. 点击「完成」→ 关闭模态，图片保留在结果页面
- **涉及页面**: StudioGenesisForm
- **备注**: 无

## 四、服装详情组图 (Clothing Studio)

### 11. [feat] 模特试穿 — 默认提示词与智能品类判断
- **优先级**: P1
- **描述**:
  用户未输入提示词但已上传产品图和模特图时：
  - 自动使用默认提示词：「将这件衣服穿到这个模特身上」
  - AI 视觉分析产品图，自动判断服装品类：
    - 上衣类（衬衫、T恤、外套等）→ 默认半身图
    - 下装类（长裤、短裤、半裙等）→ 默认全身图
  - 用户手动输入提示词则覆盖默认逻辑
- **涉及页面**: ModelTryOnTab
- **备注**: 需要 AI 视觉分析 API（Chat API 图像分类）

### 12. [feat] 基础图集 — 3D立体效果图三视角
- **优先级**: P1
- **描述**:
  选择「3D立体效果图」后，默认自动生成 3 张模特穿着图：
  - 正面图 — 模特正面穿着展示
  - 背面图 — 模特背面穿着展示
  - 侧面图 — 模特侧面穿着展示
  - 三张统一模特、统一风格、统一背景
- **涉及页面**: BasicPhotoSetTab
- **备注**: 无

### 13. [feat] 基础图集 — 人台图真实材质保留
- **优先级**: P1
- **描述**:
  人台图生成要求：
  - 严格保留衣服原始材质和外观：颜色、款式、剪裁、纹理、面料质感不做改变
  - 真实光影效果：模拟摄影棚/自然光下的高光、阴影、面料反光
  - 本质是换场景/换人台展示，不是重新设计衣服
- **涉及页面**: BasicPhotoSetTab
- **备注**: 主要是 prompt 工程优化

- **优先级**: P1
- **描述**:
  - 每张生成图片右上角显示单独保存/下载按钮
  - 点击图片 → 放大预览（背景虚化）→ 底部弹出「是否重新生成此图？」
